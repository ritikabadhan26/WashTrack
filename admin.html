# WashTrack — Hostel Laundry Portal A simple, free laundry pickup & tracking web app built for college hostellers. Students place an order with item counts, staff move it through the wash cycle, and everyone can see live status — no payments, no pricing, just pickup tracking. ## Tech Stack - **Backend:** Flask (Python) - **Database:** SQLite (via the built-in `sqlite3` module) - **Frontend:** Jinja2 templates + Tailwind CSS (CDN) ## Project Structure ``` washtrack/ ├── app.py # Flask routes ├── database.py # SQLite helper functions ├── schema.sql # Database schema (orders + order_items) ├── requirements.txt └── templates/ ├── base.html # Shared layout, header, flash messages ├── index.html # Student order form + live status board └── admin.html # Staff dashboard (grouped by pickup slot) ``` ## Setup & Run 1. **Create a virtual environment (recommended)** ```bash python3 -m venv venv source venv/bin/activate # Windows: venv\Scripts\activate ``` 2. **Install dependencies** ```bash pip install -r requirements.txt ``` 3. **Run the app** ```bash python app.py ``` The database file `washtrack.db` is created automatically on first run (via `database.ensure_db_exists()` in `app.py`), using `schema.sql`. 4. **Open in your browser** - Student view: http://127.0.0.1:5000/ - Staff dashboard: http://127.0.0.1:5000/admin ## Resetting the Database Delete `washtrack.db` and restart the app — it will be recreated automatically with fresh, empty tables. ## Features - **Student Dashboard (`/`)** — a friendly form to submit a pickup request (name, room, phone, pickup slot, and per-item counts for Shirts, Pants, Bedding, Socks, Others) plus a live board of all active hostel orders. - **Staff Dashboard (`/admin`)** — quick metric cards (Pending Pickup, Washing, Ready for Collection, Today's Orders) and orders grouped by pickup slot, each with a one-click button to advance its status: `Pending Pickup → Washing → Ready for Collection → Collected`. - **Item-level tracking** — every order stores exact counts per clothing type so nothing gets lost in the wash. - 100% free for residents — no payment or pricing logic anywhere.
